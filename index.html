Ron Swanson Quote Generator

(4 different AJAX requests types)

Quote Goes Here... Click on any of the buttons!

Quotes are generated with 4 different types of AJAX requests to get a random quote from fictional character - Ron Swanson
XHR is a vanillaJS XMLHttpRequest to get the data from the api
Fetch uses promises - .then and .catch for errors and req.json() for parsing the response
jQuery uses build-in methods .done and .fail for errors, parsing is done automatically using $.getJSON function
Axios uses promises; axios.get(url) returns JSON object so no parsing is needed

Random dog generator

This is a random dog picuture app

Dog photos are uploaded using dog.ceo api
Site uses vanillaJS XMLHttpRequest to get the data from the api
Response is parsed using JSON.parse and dog photo img tag source is updated with the API property
It is AJAX request as the code waits for the request to be ready and performed without errors (XHR.readyState == 4 && XHR.status == 200)
